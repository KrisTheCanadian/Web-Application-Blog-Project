Inspired by [@CoreyMSchafer](https://github.com/CoreyMSchafer) 's Flask Tutorials : [Flask Web Application Tutorial](https://www.youtube.com/playlist?list=PL-osiE80TeTs4UjLw5MM6OjgkjFeUxCYH)

- Pillow
- flask_login
